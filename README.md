# HumanFlow-AI
SaaS de simulação de conversas humanas com IA.

## Stack
- Node.js (Backend)
- Next.js (Frontend)
- GPT (OpenAI)
- Evolution API (WhatsApp)
- PostgreSQL
- Redis (Workers)

## Sistema
- Instâncias WhatsApp
- Agentes IA com personalidade
- Motor de simulação social
- Dashboard em tempo real
